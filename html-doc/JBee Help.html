JBee Help

    

Welcome to the help pages of JBee.

JBee is a utility for finding, installing and executing java applets. It's novel design will give you a new kind of 'net experience: Applet surfing. We hope you too will find out that this is the best way to run Java applets. We recommend every first time user to scan these pages to get the basic knowledge to gain maximum JBee-joy!

The most important components of JBee are: the Menu, the Search page and the Admin page.

Use the Menu to start applets, the Search page to find and install applet links and the Admin page to organise and manage applet links. JBee will allow you to start all applets in a folder simultaneously. To create a virtual Java desktop, just put all the applets you want in a folder and execute it! JBee will remember the position and size of each applet.
